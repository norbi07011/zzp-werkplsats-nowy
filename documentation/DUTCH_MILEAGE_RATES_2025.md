# Nederlandse Kilometervergoeding 2025

## Belastingdienst Officiële Tarieven

### Voor Zakelijke Ritten (Business Trips)

| Voertuig Type      | Tarief per km | Eigendom      |
| ------------------ | ------------- | ------------- |
| **Auto (bedrijf)** | €0,23/km      | Bedrijfsauto  |
| **Auto (privé)**   | €0,19/km      | Privé auto    |
| **Motor**          | €0,21/km      | Privé/Bedrijf |
| **Fiets**          | €0,27/km      | Privé/Bedrijf |
| **E-bike**         | €0,27/km      | Privé/Bedrijf |
| **Scooter**        | €0,21/km      | Privé/Bedrijf |

### Belastingvrije Vergoeding

**Maximum belastingvrij bedrag per jaar: €3.000**

Dit betekent:

- Als werkgever mag je tot €3.000 per jaar belastingvrij vergoeden aan werknemers
- Als ZZP'er kun je deze kosten aftrekken tot €3.000 belastingvrij
- Bedragen boven €3.000 worden als inkomen belast

### Berekening Voorbeeld

**Auto (privé) - 8.500 km zakelijk per jaar:**

```
8.500 km × €0,19 = €1.615 belastingvrij
```

**Fiets - 2.000 km woon-werk per jaar:**

```
2.000 km × €0,27 = €540 belastingvrij
```

## Werkelijke Autokosten (NIBUD 2025)

Volgens het Nibud zijn de werkelijke kosten van een auto hoger dan de vergoeding:

| Auto Klasse             | Km per jaar | Kosten per maand | Km-prijs totaal | Km-prijs variabel |
| ----------------------- | ----------- | ---------------- | --------------- | ----------------- |
| **Miniklasse**          | 8.500       | €356             | 50,3 ct/km      | 22,3 ct/km        |
| **Compacte klasse**     | 9.000       | €419             | 55,9 ct/km      | 24,0 ct/km        |
| **Kleine middenklasse** | 11.000      | €554,50          | 60,5 ct/km      | 23,7 ct/km        |
| **Middenklasse**        | 11.000      | €703,50          | 80,4 ct/km      | 30,6 ct/km        |

### Vaste Kosten vs Variabele Kosten

**Vaste kosten (per maand):**

- Afschrijving
- Verzekering (WA, Casco)
- Motorrijtuigenbelasting (wegenbelasting)
- Onderhoud (basis)

**Variabele kosten (per km):**

- Brandstof/Elektriciteit
- Extra afschrijving door gebruik
- Onderhoud en reparatie (wear & tear)
- Banden

## Woon-Werkverkeer (Commute)

Voor woon-werkverkeer gelden andere regels:

### Reisaftrek Openbaar Vervoer

- Maximale reisaftrek per kilometer (enkele reis)
- Afhankelijk van afstand woning-werk

### Eigen Vervoer (Auto/Motor/Fiets)

- **GEEN** directe fiscale aftrek voor woon-werkverkeer met eigen vervoer
- Wel mogelijk als zelfstandige via kilometeradministratie
- Binnen "werkkostenregeling" kan werkgever vergoeden

## Verschillen per Type

### Auto vs Fiets

| Aspect            | Auto (privé) | Fiets       |
| ----------------- | ------------ | ----------- |
| Tarief            | €0,19/km     | €0,27/km    |
| Max belastingvrij | €3.000/jaar  | €3.000/jaar |
| Werkelijke kosten | 50-80 ct/km  | ~5 ct/km    |
| Milieuvoordeel    | Geen         | Ja          |

### Waarom Fiets Hogere Vergoeding?

- Beloning voor milieuvriendelijke keuze
- Stimulering gezonde levensstijl
- Lagere werkelijke kosten = hogere netto opbrengst

## ZZP'ers Specifiek

### Wat Kun Je Aftrekken?

✅ **WEL aftrekbaar:**

- Zakelijke ritten (klant bezoeken, vergaderingen)
- Ritten naar tijdelijke werklocaties
- Netwerkevents en zakelijke bijeenkomsten
- Ritten voor inkopen/materiaal

❌ **NIET aftrekbaar:**

- Vaste woon-werkverkeer (kantoor naar huis)
- Privé ritten
- Vakantie ritten

### Administratie Vereisten

Voor belastingaftrek heb je nodig:

1. **Kilometeradministratie** met:

   - Datum
   - Start- en eindbestemming
   - Aantal kilometers
   - Doel van de rit (zakelijk)
   - Kenteken voertuig

2. **Bewijs** van zakelijk karakter:
   - Afspraken kalender
   - Facturen locatie
   - Projectadministratie

## Motor Specifiek

### Motorrijtuigenbelasting

- Hoger dan auto in veel gevallen
- Afhankelijk van cilinderinhoud
- Per kwartaal te betalen

### Verzekering

- WA verplicht
- Casco optioneel
- Meestal duurder dan auto per km

### Vergoeding

€0,21/km is midden tussen auto en fiets

- Goedkoper dan auto qua brandstof
- Duurder dan fiets qua onderhoud
